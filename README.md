# A Funeral Star Website

Static site for `afuneralstar.com`, built from `WEBSITE_BRIEF.md`.

## Files

- `index.html` - full single-page band site
- `assets/css/styles.css` - responsive visual system
- `assets/js/site.js` - mobile nav and YouTube playlist modal
- `assets/img/` - copied or generated web assets only

## Deployment

Upload the contents of this folder to the GoDaddy web root, usually `public_html`.
